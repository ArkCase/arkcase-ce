You may find it useful to create a local host name 'arkcase-ce.local' to the IP address assigned to this VM. To find the IP address, run this command from the folder where the Vagrantfile is located: "vagrant ssh -c ifconfig. (Note, the vagrant user's password is vagrant). Look for the IP address associated with the enp0s8 network interface.
